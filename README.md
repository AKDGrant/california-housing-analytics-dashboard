# California Housing Analytics Dashboard

## Overview

This project analyzes the California Housing dataset using Python, Pandas, and Matplotlib to identify factors that influence housing prices.

The goal was to perform exploratory data analysis (EDA), visualize relationships between variables, and generate insights from over 20,000 housing records.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-learn
* Google Colab

## Dataset

The project uses the California Housing dataset provided by Scikit-learn, containing 20,640 records and housing-related features such as:

* Median Income
* House Age
* Average Rooms
* Population
* Average Occupancy
* Latitude & Longitude
* Median House Value

## Analysis Performed

### Dashboard Visualizations

Created a multi-panel dashboard analyzing:

* Income vs House Value
* House Age vs House Value
* Population vs House Value
* Average Rooms vs House Value

### Correlation Analysis

Built a correlation heatmap to identify relationships between housing features and property values.

## Key Findings

* Median income showed the strongest positive relationship with house values.
* Population alone was not a strong predictor of housing prices.
* Housing characteristics such as room count showed moderate relationships with property values.
* Several outliers were identified that may warrant additional investigation.
